# Customer Purchase Prediction System

A full-stack machine learning application that predicts whether a customer is likely to purchase a product using customer demographics and browsing behavior.

## Features

- Customer purchase prediction
- Four ML models:
  - Logistic Regression
  - Decision Tree
  - Random Forest
  - KNN
- MongoDB Atlas integration
- Real-time prediction logging
- Save customer records
- Model retraining using new data
- Interactive Flask web application

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- MongoDB Atlas
- HTML
- CSS
- JavaScript
